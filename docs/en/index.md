## Language Model

`README.md`, community files without a language suffix, and the default docs
path are German. English alternatives use `.en.md` or live under `docs/en/`.
GitHub does not automatically translate the regular repository view by visitor
language, so language links and explicit files are the repository-level i18n
mechanism.
