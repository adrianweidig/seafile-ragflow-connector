# RAGFlow Template

The connector looks for one dataset named by `RAGFLOW_TEMPLATE_DATASET_NAME`, default
`connector_template`.

The template is used only for dataset creation. Existing datasets keep their own
current RAGFlow settings.

## Create Payload Whitelist

- `avatar`
- `description`
- `embedding_model`
- `permission`
- `chunk_method`
- `parser_config`
- `parse_type`
- `pipeline_id`

`name` is always generated by the connector.

Built-in chunking (`chunk_method`, `parser_config`) and ingestion pipeline mode
(`parse_type`, `pipeline_id`) must not be mixed.
