# Code of Conduct

🌐 Languages: [Deutsch](CODE_OF_CONDUCT.md) | **English**

This project should support factual and respectful collaboration. Everyone is
invited to describe problems clearly, challenge technical decisions, and suggest
improvements.

## Expected Behavior

- Communicate respectfully and concretely.
- Support technical objections with observable facts.
- Keep discussions focused on the issue or pull request.
- Handle operational data, logs, and security information carefully.
- Help new contributors understand the project boundaries.

## Unacceptable Behavior

- Personal attacks, harassment, or demeaning language.
- Publishing secrets, private data, or confidential logs.
- Pressuring others to perform risky productive changes without review.
- Repeatedly ignoring documented security or review guidance.

## Enforcement

Maintainers may edit, hide, or close comments when they harm collaboration or
project safety. For serious or repeated violations, maintainers may restrict
further interaction in the repository.

This Code of Conduct is intentionally concise and project-specific. It
complements [CONTRIBUTING.en.md](CONTRIBUTING.en.md) and
[SECURITY.en.md](SECURITY.en.md).
